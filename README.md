# WealthOS

Production-ready foundation for the WealthOS SaaS application.

## Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase
- React Hook Form
- Zod
- Lucide React
- Recharts
- ESLint
- Prettier

## Scripts

- `npm run dev`
- `npm run build`
- `npm run lint`

## Environment

Copy `.env.example` to `.env.local` and provide your Supabase values.
